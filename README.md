# Hariprasad AP — Professional Developer Portfolio

A premium, optimized, and modern developer portfolio showcasing academic qualifications, core skills, certifications, and structural "Proof of Work" projects. 

Live version features a custom **Warm Sand / Terracotta** color story, high-performance vanilla styling, and a seamless visual theme engine.

---

## Key Features

*   **Zero-Flash Dark Theme default**: Out-of-the-box Dark Mode styling with an inline head script to prevent flash of unstyled content (FOUC).
*   **Aesthetic & Modern Layout**: Styled with custom typography (Crimson Text & Work Sans) and a translucent, glassmorphic header overlay (`backdrop-filter`).
*   **Responsive Proof of Work Grid**: Flexbox and CSS Grid layout adapting seamlessly across mobile, tablet, and widescreen monitors.
*   **Timeline Education Cards**: Sleek stacked panels detailing MCA and BCA coursework with custom vector graduation badges.
*   **Performance Audited**: Stripped of heavy CSS libraries, unused dependencies (like Font Awesome CDN requests), and unlinked binary documents for instant loading.

---

## Technology Stack

*   **Markup**: Vanilla HTML5 (semantic layout hierarchy, responsive metadata, SEO optimized)
*   **Styling**: Vanilla CSS3 (Custom properties, theme tokens, fluid typography, smooth variables-based theme transitions)
*   **Interactive Engine**: Vanilla JavaScript (IntersectionObserver API for scroll animation triggers, active section tracking, localStorage theme sync)
*   **Icon Assets**: Inline SVGs & CoreUI Brand Icon pack (for LeetCode branding)
